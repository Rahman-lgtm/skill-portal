HTML Basics – Lesson 1

HTML কি?

HTML ৰ সম্পূৰ্ণ নাম হৈছে Hyper Text Markup Language।

HTML হৈছে এটা ভাষা যাক ব্যৱহাৰ কৰি Website বনোৱা হয়।

যেতিয়া তুমি Google, YouTube, বা অন্য কোনো Website খোলা, সেইবোৰ Website বেছিভাগ সময় HTML ব্যৱহাৰ কৰি বনোৱা হয়।

👉 HTML = Website বনোৱাৰ ভিত্তি

যেনে এটা ঘৰ বনাবলৈ প্ৰথমে ইট, বালি, চিমেণ্ট লাগে। তেনেই Website বনাবলৈ HTML লাগে।

Website কেনেকৈ বনোৱা হয়?

Website বনাবলৈ সাধাৰণতে ৩টা জিনিস লাগে —

  • HTML – Website ৰ গঠন বনাই
  • CSS – Website ধুনীয়া কৰে
  • JavaScript – Website চলন্ত (Interactive) কৰে

HTML ক'ত লিখা হয়?

HTML লিখিবলৈ তুমি এইবোৰ ব্যৱহাৰ কৰিব পাৰা —

  • Notepad
  • Notepad++
  • VS Code
  • Sublime Text

সৰু ল'ৰা-ছোৱালীৰ বাবে Notepad দিয়াই আৰম্ভ কৰা ভাল।

HTML File কেনেকৈ বনাব?

Step by step —

  1. Computer ত Notepad খুলা
  2. HTML code লিখা
  3. File Save কৰা
  4. File name দিয়া
index.html

গুৰুত্বপূর্ণ কথা

👉 File ৰ শেষত .html থাকিব লাগিব।

HTML Tag কি?

HTML ত আমি Tag ব্যৱহাৰ কৰো।

Tag মানে হৈছে নির্দেশনা।

Tag browser ক কয় —

  • এইটো heading
  • এইটো paragraph
<tagname> </tagname>
Example
<p>এইটো এটা paragraph</p>

<p> হৈছে start tag

</p> হৈছে end tag

HTML ৰ প্ৰথম Program

<html> <head> <title>My First Website</title> </head> <body> Hello World </body> </html>

Heading Example

<h1>Big Heading</h1> <h2>Medium Heading</h2> <h3>Small Heading</h3>

Paragraph Example

<p>মই HTML শিকি আছো</p>

Line Break

<p> মোৰ নাম Rahul <br> মই HTML শিকো </p>

HTML Comment

<!-- এইটো comment --> <h1>Hello</h1>

Practice Example

<html> <head> <title>My Website</title> </head> <body> <h1>Welcome</h1> <p>মোৰ প্ৰথম Website</p> <p>মই HTML শিকি আছো</p> </body> </html>
